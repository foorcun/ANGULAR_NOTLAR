# create project
##Clone the template
```.sh
npx degit chromaui/intro-storybook-angular-template taskbox
```

# Install dependencies
```.sh
cd taskbox

npm install
```

# run to test
## Start the component explorer on port 6006:
```.sh
ng run taskbox:storybook
```

## Run the frontend app proper on port 4200:
```.sh
ng serve
```
